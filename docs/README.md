# 자동차 경주

## 구현할 기능 목록

1. (입력) 경주할 차 이름 입력

   - 유효성 검사
   - 쉼표로 구분
   - 5자 이하
   - only eng

2. (입력) 시도할 횟수

   - 루프의 횟수가 될 것임

3. 자동차 객체 만들기 -> 인스턴스 배열 생성
   new Car();

   - 이름 초기화
   - 전진 여부 (0~9 사이의 수 중 4이상)
   - 이동 출력 Line
   - 전진 횟수

4. (2) 시도 횟수만큼 반복하여 자동차 이동 상태 출력 (loop)

   - '-' 하나씩 추가해서 보여주기

5. 마지막에 최종 우승자 보여주기
   - 한 명 이상일 수 있다.
